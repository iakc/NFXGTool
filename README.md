## 这个软件有什么用

南方稀贵金属交易所行情客户端可以查看白银及各种小金属的行情走势，但是对于某些品种（如氧化铕），早期的品种交易单位与后期的交易单位有所差别，导致查看大级别时K线图及MACD图变形，本软件可以把早期数据交易单位修改到和后期交易单位一致，使走势图便于观看。

> 南方稀贵官方网站现已不提供行情客户端下载，你可以在其它下载网站搜索并下载。
